Matrix/Element 


До уваги Рівняни!

Підключайтесь до зашифрованого чат-серверу (Matrix), розташованому у Рівне. Працює в локальній мережі вашого провайдера. Ви зможете в ній обмінюватися повідомленнями з близькими та друзями, у випадку, якщо інтернет зникне.

  • Для десктопа: https://chat.rivne.dcomm.net.ua/
  • Для смартфона/app: ​​https://element.io/get-started#download
  • Для app: під час налаштування використовуйте наступну назву домашнього сервера: https://matrix.rivne.dcomm.net.ua

Як розпочати:

  • Створіть обліковий запис
  • Створіть/приєднайтесь до публічної кімнати
  • Створіть приватну кімнату та відправте посилання друзям

Як це працює:

  • усі особисті та групові чати наскрізно зашифровані
  • користувачі створюють приватні кімнати та обмінюються своїми ID
  • мережа локалізована у Рівне, ніхто ззовні не зможе нею користуватися

Поділіться з друзями та близькими, створюйте ПРИВАТНІ кімнати для обміну повідомленнями.
